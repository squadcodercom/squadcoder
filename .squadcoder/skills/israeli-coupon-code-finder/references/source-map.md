# Israeli Coupon Source Map (quick reference)

Domains are written without a scheme on purpose. The skill is methodology-only: it never stores actual codes. Search these source types live and verify every hit before presenting it.

## Source types, in the order to try them

| # | Source type | Why it matters | What to search / check | Verified examples |
|---|-------------|----------------|------------------------|-------------------|
| 1 | Coupon aggregators | Primary place public codes are indexed | Per-store code pages, dated posts | couponim.co.il, and other Hebrew "קופונים" sites the search surfaces |
| 2 | Store-direct levers | Most reliable, lowest scam risk | First-order newsletter/SMS code, app-only code, abandoned-cart email, loyalty-club price | the store's own site and app |
| 3 | Cashback platforms | Can stack on or beat a code | Store's rebate rate, whether it stacks with a code | cashback.co.il |
| 4 | Credit-card benefit hubs | A cardholder discount can beat any public code | The shopper's own card issuer's benefits hub | max.co.il/benefits |
| 5 | Gift-card stacking | Effective discount paid at checkout | A discounted gift card for the store | buyme.co.il |
| 6 | Consumer / employee clubs | Member pricing the shopper may already have | Eligibility (career soldier, retiree, works council) | hvr.co.il |
| 7 | Telegram coupon channels | Fast-rotating codes | Search live; never assume a specific channel | search-only, do not name a channel |
| 8 | Seasonal windows | Timing beats a single code | Is a known sale near? | Black Friday, Cyber Monday, Singles Day, end-of-season, holiday sales |

## Hebrew search templates (swap in the store name)

```
קוד קופון <חנות> <שנה>
קופון <חנות> הנחה
<חנות> מבצעים קוד הנחה
<store-name> coupon code israel
<store-name> promo code
```

## Verification rubric (per candidate)

- Expiry date (drop expired).
- Minimum-cart threshold.
- New-customer-only / first-order-only flag (the most common false positive).
- Category / brand exclusions.
- Single-use vs reusable; stacks with sale items or not.
- Posting date present (no date - Low confidence).

## Output (single ranked table)

`Code | Source (link) | Date seen | Discount | Conditions | Confidence`, best first, plus a "try this first" pick, plus a one-line note if a cashback rebate, card benefit, or discounted gift card beats the top code, plus the verify-at-checkout reminder.

## Legal backstop

A remote (online) purchase can be cancelled within 14 days of receiving the product or the transaction-details document; the cancellation fee is capped at the lower of 5% of the price or 100 NIS. People with disabilities, shoppers 65 or older, and new immigrants get 4 months. See the Kol-Zchut remote-sale cancellation page (full URL in SKILL.md).
