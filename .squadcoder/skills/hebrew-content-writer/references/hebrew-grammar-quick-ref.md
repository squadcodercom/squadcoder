# Hebrew Grammar Quick Reference for Content Writers

## Binyanim (Verb Patterns)

Hebrew has 7 verb patterns (binyanim), each with a different semantic role:

| Binyan | Pattern | Meaning | Example |
|--------|---------|---------|---------|
| Pa'al (Kal) | פָּעַל | Simple active | כָּתַב (wrote) |
| Nif'al | נִפְעַל | Simple passive / reflexive | נִכְתַּב (was written) |
| Pi'el | פִּעֵל | Intensive active | דִּבֵּר (spoke), סִפֵּר (told) |
| Pu'al | פֻּעַל | Intensive passive | דֻּבַּר (was spoken), סֻפַּר (was counted/told) |
| Hif'il | הִפְעִיל | Causative active | הִכְתִּיב (dictated) |
| Huf'al | הֻפְעַל | Causative passive | הֻכְתַּב (was dictated) |
| Hitpa'el | הִתְפַּעֵל | Reflexive / reciprocal | הִתְכַּתֵּב (corresponded) |

**For formal writing:** Use passive binyanim (Nif'al, Pu'al, Huf'al) more frequently.
**For casual writing:** Prefer active binyanim (Pa'al, Pi'el, Hif'il).

## Register Examples

### Formal (Legal/Government)
```
בהתאם להוראות סעיף 5 לחוק, מוסמך המנהל להורות על ביטול הרישיון.
(In accordance with the provisions of Section 5 of the law, the Director is authorized to order the revocation of the license.)
```

### Business (Corporate/Professional)
```
אנו שמחים להודיע על השקת המוצר החדש שלנו, המיועד לייעל את תהליכי העבודה בארגון.
(We are pleased to announce the launch of our new product, designed to streamline organizational workflows.)
```

### Casual (Social Media/Chat)
```
יאללה, המוצר החדש שלנו סוף סוף בחוץ! בואו לבדוק 🚀
(Yalla, our new product is finally out! Come check it out)
```

## Ktiv Maleh vs. Ktiv Chaser

Ktiv Maleh (full spelling) adds vav and yod to indicate vowels. Always use Ktiv Maleh in modern content:

| Ktiv Chaser (avoid) | Ktiv Maleh (use) | Word |
|---------------------|------------------|------|
| תכנה | תוכנה | software (tochnah) |
| שרות | שירות | service (sherut) |
| קדם | קודם | before (kodem) |
| תכנית | תוכנית | plan / program (tochnit) |
| פרוט | פירוט | itemization (perut) |

## Common Smichut (Construct State) Forms

| Base form | Smichut form | Example |
|-----------|-------------|---------|
| בית (bayit) | בית (beit) | בית ספר (school) |
| דבר (davar) | דבר (dvar) | דבר תורה (Torah teaching) |
| שם (shem) | שם (shem) | שם משפחה (family name) |
| ספר (sefer) | ספר (sifrei) | ספרי ילדים (children's books, pl.) |

**Rule:** In smichut, only the LAST noun takes the definite article "ha-":
- Correct: בית הספר (beit ha-sefer)
- Wrong: הבית הספר (ha-beit ha-sefer)

## Gender-Inclusive Writing Patterns

### Pattern 1: Infinitive Construction
Replace gendered imperatives with infinitive forms:
- Instead of: לחץ כאן (click here, masculine)
- Use: יש ללחוץ כאן (one should click here)

### Pattern 2: Passive Voice
- Instead of: אתה צריך למלא את הטופס (you need to fill the form, m.)
- Use: יש למלא את הטופס (the form should be filled)

### Pattern 3: Collective Nouns
- Instead of: המשתמשים מרוצים (the users are satisfied, m.)
- Use: קהילת המשתמשים מרוצה (the user community is satisfied)

### Pattern 4: Slash Notation (when other patterns are awkward)
- היזמים/ות (the entrepreneurs, m/f)
- המפתחים/ות (the developers, m/f)
